	Install client

2.Put mods' file into path below
C:\Users\"USERNAME"\AppData\Roaming\.minecraft\mods
(if you installed this pack before, overwrite the file when ask)

3.Start minecraft launcher

4.Add Ram if necessary
	follow the final steps listed in "profile_and_add_Ram.jpg" to add Ram
	
5.Select profile with forge-1.12.2-14.23.5.2847
	if no matching profile to start, follow the steps listed in "profile_and_add_Ram.jpg"

6.You are ready to play the game
